initialize

@JvmStatic fun initialize(context: Context, onGetTokenCallback: OnGetTokenCallback, onResultListener: OnResultListener<EThree>): Unit

Current method allows you to initialize EThree helper class. To do this you should provide onGetTokenCallback that must return Json Web Token string representation with identity of the user which will use this class. In onResultListener you will receive instance of EThree class or an Throwable if something went wrong.
