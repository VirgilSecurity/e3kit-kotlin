fun lookupPublicKeys(identity: String): Result<LookupResult>

Retrieves user public key from the cloud for encryption/verification operations.

Searches for public key with specified identity and returns map of String -> PublicKey in onResultListener callback or Throwable if something went wrong.

PublicKeyNotFoundException will be thrown if public key wasn't found.

Can be called only if private key is on the device, otherwise PrivateKeyNotFoundException exception will be thrown.
