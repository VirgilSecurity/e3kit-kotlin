resetPrivateKeyBackup

@JvmOverloads fun resetPrivateKeyBackup(password: String? = null): Completable

Deletes the user's private key from Virgil's cloud.

Deletes private key backup using specified password and provides onCompleteListener callback that will notify you with successful completion or with a Throwable if something went wrong.

Can be called only if private key is on the device otherwise PrivateKeyNotFoundException exception will be thrown.
