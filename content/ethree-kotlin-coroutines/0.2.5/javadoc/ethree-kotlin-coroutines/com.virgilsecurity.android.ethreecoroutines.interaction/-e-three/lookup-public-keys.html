lookupPublicKeys

fun lookupPublicKeys(identities: List<String>): Deferred<Map<String, PublicKey>>

Retrieves user public keys from the cloud for encryption/verification operations.

Searches for public keys with specified identities and returns list of PublicKey in onResultListener callback or Throwable if something went wrong.

PublicKeyNotFoundException will be thrown for the first not found public key.

Can be called only if private key is on the device otherwise PrivateKeyNotFoundException exception will be thrown.
