findUsers

fun findUsers(identities: List<String>, forceReload: Boolean = false, checkResult: Boolean = true): Result<FindUsersResult>

Retrieves users Cards from the Virgil Cloud or local storage if exists.

To start execution of the current function, please see Result description.

Parameters

identities - Array of identities to find.

forceReload - Will not use local cached cards if true.

checkResult - Checks that cards for all identities were found if true.

Exceptions

FindUsersException - (FindUsersException.Description.CARD_WAS_NOT_FOUND) If card duplicates was found or at least one card was not found.

Return
FindUsersResult with found users.
