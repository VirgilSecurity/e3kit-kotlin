unregister

@Synchronized fun unregister(): Completable

Revokes the public key for current identity in Virgil's Cards Service. After this operation you can call EThreeCore.register again.

To start execution of the current function, please see Completable description.

Exceptions

EThreeException - if there's no public key published yet, or if there's more than one public key is published.
