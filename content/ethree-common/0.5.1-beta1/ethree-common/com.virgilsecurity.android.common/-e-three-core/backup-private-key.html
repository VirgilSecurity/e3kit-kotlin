backupPrivateKey

fun backupPrivateKey(password: String): Completable

Encrypts the user's private key using the user's password and backs up the encrypted private key to Virgil's cloud. This enables users to log in from other devices and have access to their private key to decrypt data.

Encrypts loaded from private keys local storage user's Private key using Public key that is generated based on provided password after that backs up encrypted user's Private key to the Virgil's cloud storage.

Can be called only if private key is on the device otherwise PrivateKeyNotFoundException exception will be thrown.

To start execution of the current function, please see Completable description.
