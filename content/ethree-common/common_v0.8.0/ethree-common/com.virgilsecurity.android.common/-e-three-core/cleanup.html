cleanup

fun cleanup(): Unit
  • ! WARNING ! If you call this function after register without using backupPrivateKey then you loose private key permanently, as well you won't be able to use identity that was used with that private key no more.

Cleans up user's private key from a device - call this function when you want to log your user out of the device.

Can be called only if private key is on the device otherwise EThreeException.Description.MISSING_PRIVATE_KEY exception will be thrown.
